Bayview is a web-based real estate company created with a user-friendly and easy-to-navigate interface, offering various features where users can effortlessly search for properties with detailed descriptions. TransUPN is a web-based bus database system application, where the admin can perform CRUD (Create, Read, Update, Delete) operations on every existing data directly connected to the database. The admin has full control over bus data, driver information, conductor details, as well as salary records for drivers and conductors, all in accordance with the implemented system.

Skill.

HTML (Hypertext Markup Language) is a markup language used to create the content structure on web pages, allowing the use of tags and attributes to define elements such as text, images, and links.

CSS (Cascading Style Sheets) is a stylesheet language used for describing the look and formatting of a document written in HTML. It enables web designers to control the visual presentation of elements on a webpage, enhancing its aesthetics and user experience.
